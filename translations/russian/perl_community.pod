=head1 Сайты сообщества

X<perl.org>
X<веб-сайты; perl.org>

Домашняя страница Perl на U<http://www.perl.org/> ссылается на документацию
Perl, исходный код, учебные пособия, списки рассылки и некоторые важные проекты
сообщества. Если вы новичок в Perl, то список рассылки Perl beginners это
дружелюбное место для вопросов новичков и получения точных и полезных ответов.
Смотрите U<http://learn.perl.org/faq/beginners.html>.

Дом разработки Perl на U<http://dev.perl.org/>, которая ссылается на
соответствующие ресурсы по разработке Perl 5 и Perl 6N<Также загляните на
U<http://www.perl6.org/>>.

X<perl.com>
X<веб-сайты; perl.com>

Perl.com публикует статьи и учебные пособия о Perl и его культуре. Его архивы
уходят в 20ый век. Смотрите U<http://www.perl.com/>.

X<cpan.org>
X<веб-сайты; cpan.org>
X<MetaCPAN>
X<веб-сайты; MetaCPAN>

Основное расположение CPAN (L<cpan>) на сайте U<http://www.cpan.org/>, хотя
опытные пользователи проводят больше времени на U<http://search.cpan.org/>.
Этот основной пункт распространения программ, полезного и свободного Perl кода,
является неотъемлемой частью Perl сообщества.
MetaCPAN (U<https://metacpan.org/>) - это недавно появившаяся альтернатива
веб-интерфейса к CPAN.

X<PerlMonks>
X<веб-сайты; PerlMonks>

PerlMonks на U<http://perlmonks.org/> - это сайт сообщества посвящённый
обсуждению программирования на Perl. Его одиннадцатилетняя история делает его
одним из наиболее почтенных сайтов вопросов и ответов для любых языков
программирования.

X<blogs.perl.org>
X<веб-сайты; blogs.perl.org>

Некоторые сайты сообщества представляют новости и комментарии.
U<http://blogs.perl.org/> - это свободная блог платформа, открытая для любого
участника сообщества Perl.

X<Enlightened Perl Organization>
X<Planet Perl>
X<Planet Perl Iron Man>
X<веб-сайты; Planet Perl>
X<веб-сайты; Planet Perl Iron Man>

Другие сайты агрегирующие размышления Perl хакеров, включают
U<http://perlsphere.net/>, U<http://planet.perl.org/>, и
U<http://ironman.enlightenedperl.org/>. Последний является частью инициативы
Организации Просвещения Perl (U<http://enlightenedperl.org/>) для увеличения
числа и повышения качества Perl публикаций в сети.

X<Perl Buzz>
X<веб-сайты; Perl Buzz>
X<Perl Weekly>
X<веб-сайты; Perl Weekly>

Perl Buzz (U<http://perlbuzz.com/>) собирает и перепечатывает некоторые
наиболее интересные и полезные новости о Perl на регулярной основе. Perl Weekly
(U<http://perlweekly.com/>) представляет еженедельные новости из мира Perl.

=head1 Сайты Разработки

X<RT>
X<Best Practical>

Best Practical Solutions (U<http://bestpractical.com/>) поддерживает
инсталляцию их популярной системы отслеживания запросов - RT, как для авторов
CPAN, так и для разработки Perl 5 и Perl 6. Каждый дистрибутив CPAN имеет свою
собственную очередь RT, имеющую ссылку на C<search.cpan.org> и доступную на
сайте U<http://rt.cpan.org/>. Perl 5 и Perl 6 имеют отдельную систему запросов
на сайте U<http://rt.perl.org/>.

X<Perl 5 Porters>
X<p5p>

Список рассылки Perl 5 Porters (или I<p5p>) - это точка фокуса разработки
Perl 5. Смотрите U<http://lists.cpan.org/showlist.cgi?name=perl5-porters>.

X<TPF>
X<The Perl Foundation>
X<TPF; wiki>
X<веб-сайты; TPF wiki>

Фонд Perl (U<http://www.perlfoundation.org/>) поддерживает wiki для всего, что
относится к Perl 5. Смотрите U<http://www.perlfoundation.org/perl5>.

X<Github>
X<gitpan>
X<веб-сайты; gitpan>

Многие Perl хакеры используют GitHub (U<http://github.com/>) для хостинга их
проектовN<... включая исходный код данной книги на
U<http://github.com/chromatic/modern_perl_book/>>. Особенно стоит посмотреть
Gitpan (U<http://github.com/gitpan/>), в котором хранятся Git репозитории с
хроникой полной истории каждого дистрибутива CPAN.

=begin tip Локальное Git зеркало

X<CPAN; C<Git::CPAN::Patch>>
X<Champoux, Yanick>

=end tip

=head1 События

X<YAPC>

Сообщество Perl проводит бесчисленное количество конференций, практических
секций, семинаров и встреч. В частности проводимое сообществом YAPC -- Ещё Одна
Perl Конференция -- это успешная, местная, недорогая модель конференций,
проводимая на нескольких материках. Смотрите U<http://yapc.org/>.

На wiki Фонда Perl перечислены другие события
U<http://www.perlfoundation.org/perl5/index.cgi?perl_events>.

X<Perl Mongers>

Сотни локальных Perl Mongers групп часто собираются для технических обсуждений
и социального общения. See U<http://www.pm.org/>.

=head1 IRC

X<IRC>
X<IRC; #moose>
X<IRC; #catalyst>
X<IRC; #perl-help>
X<IRC; #perl>

Когда Perl маньяки не могут встретиться лично, многие совместно работают и
общаются в сети через текстовую систему чата, известную как IRC. Многие из
наиболее популярных и полезных Perl проектов имеют свои IRC каналы, такие как
I<#moose> и I<#catalyst>.

Основной сервер для Perl сообщества это U<irc://irc.perl.org/>. Наиболее
полезные каналы: I<#perl-help>, для общих вопросов по программированию на Perl
и I<#perl-qa>, посвящённый тестированию и другим вопросам качества. Знайте, что
канал I<#perl> является каналом общего назначения для обсуждения всего чего
захотят обсудить его участникиN<... и, поэтому он не является основным местом
для помощи.>.
