=head1 CPAN

Z<cpan>

Perl 5 прагматичен по своей сути, кроме того всегда прагматичное Perl сообщество
расширило этот язык и сделало свою работу доступной для всего мира. Если у вас
задача, требующая решения, то есть все шансы, что кто-то уже написал и поделился
Perl кодом для неё.

X<CPAN>

Современное программирование на Perl активно использует CPAN
(U<http://www.cpan.org/>). Всеобъемлющий Сетевой Архив Perl - это система
загрузки и зеркалирования для свободно распространяемого и повторно
используемого Perl кода. Это один из крупнейших в мире, если не I<самый>
крупный, архив библиотек кода. CPAN содержит библиотеки для чего угодно: от
доступа к базам данным, до профилирующих утилит, от протоколов для почти всех
когда либо созданных сетевых устройств, до звуковых и графических библиотек, и
обёрток для разделяемых библиотек в вашей системе.

Современный Perl без CPAN это просто ещё один язык. Современный Perl вместе со
CPAN - это просто потрясающе.

X<дистрибутив>
X<модули>

CPAN зеркалирует дистрибутивы или коллекции полезного Perl кода. Каждый
дистрибутив может содержать один или несколько I<модулей> или замкнутых
библиотек Perl кода. Каждый дистрибутив занимает своё собственное пространство
имён на CPAN и предоставляет уникальные метаданные.

=begin tip CPAN Огромный, Реально Огромный

CPAN I<добавляет> сотни зарегистрированных участников и тысячи индексированных
модулей в сотнях дистрибутивах ежемесячно. Эти цифры не учитывают обновления.
В конце ноября 2011 года search.cpan.org отчитался о 9359 участниках, 101656
модулях и 23808 дистрибутивах (показав темпы роста на 11.5%, 19.4% и 14.3% с
предыдущей версии этой книги, соответственно).

=end tip

X<search.cpan.org>

CPAN, сам по себе, это только сервис зеркалирования. Авторы загружают
дистрибутивы на сайты зеркал, с которых пользователи и CPAN клиенты загружают,
конфигурируют, собирают, тестируют и устанавливают их. Система успешна как
из-за своей простоты, так из-за вклада тысяч добровольцев, которые создают на
основе этой системы распространения. В частности, развитие стандартов сообщества
позволило установить свойства и характеристики правильно сформированного
CPAN дистрибутива. Которые включают:

=over 4

=item * Стандарт автоматизированного инсталлятора CPAN.

=item * Стандарт для метаданных для описания того, что каждый дистрибутив
предоставляет и ожидает.

=item * Стандарт для документации и лицензирования.

=back

Дополнительно сервис CPAN предоставляет всестороннее автоматизированное
тестирование и отчётность для улучшения качества упаковки и корректности между
различными платформами и версиями Perl. Каждый CPAN дистрибутив имеет свою
собственную очередь заявок на U<http://rt.cpan.org/> для сообщения об ошибках и
работы с авторами. Сайты CPAN также ссылаются на предыдущие версии дистрибутива,
рейтинги модулей, аннотации документации и другое. Всё это доступно на
U<http://search.cpan.org/>.

X<CPAN; C<CPAN.pm>>
X<CPAN; C<CPANPLUS>>

Инсталляция Современного Perl включает два клиента для подключения, поиска,
загрузки, сборки, тестирования и установки CPAN дистрибутивов: CPAN.pm и
CPANPLUS. По большей части эти клиенты идентичны для базовых установок. Эта
книга рекомендует использование CPAN.pm исключительно по причине его
повсеместного наличия. На последней версии (на момент написания, 1.9800 - это
последний стабильный выпуск), установка модулей довольна проста. Запустите
клиент:

=begin screen

    $ B<cpan>

=end screen

Чтобы установить дистрибутив в клиенте:

=begin screen

    $ B<cpan>
    cpan[1]> B<install Modern::Perl>

=end screen

... или, чтобы установить непосредственно из командной строки:

=begin screen

    $ B<cpan Modern::Perl>

=end screen

X<CPAN; C<CPAN.pm>>

Инструкция Эрика Уилхельма (Eric Wilhelm) по настройке
CPAN.pmN<U<http://learnperl.scratchcomputing.com/tutorials/configuration/>>
включает отличную главу по устранению неполадок.

=begin sidebar

Несмотря на то, что CPAN клиент это базовый модуль для дистрибутива Perl 5, вам
потребуется установить стандартные утилиты разработки, такие как C<make> и
возможно C компилятор. Пользователи Windows, могут посмотреть Strawberry Perl
(U<http://strawberryperl.com/>) и Strawberry Perl Professional. Пользователи
Mac OS X могут установить XCode. Пользователям Unix и Unix-подобных систем часто
эти утилиты уже доступны (хотя пользователям Debian и Ubuntu требуется
установить пакет C<build-essential>).

=end sidebar

=head2 Инструменты управления CPAN

Если ваша операционная система предоставляет свою собственную инсталляцию Perl
5, эта версия может быть уже устаревшей или иметь свои зависимости на
специфические версии дистрибутивов CPAN. Серьёзные Perl разработчики часто
создают виртуальную стену между системным Perl и своими собственными
инсталляциями Perl. Некоторые проекты позволяющие сделать это возможным.

C<App::cpanminus> - это относительно новый CPAN клиент с преимуществом в
скорости, простоте и отсутствием необходимости в настройке. Установите его из
C<cpan App::cpanminus>, или:

=begin screen

    $ B<curl -LO http://xrl.us/cpanm>
    $ B<chmod +x cpanm>

=end screen

C<App::perlbrew> - это система управления и переключения между вашими
собственными инсталляциями множества версий и конфигураций Perl. Инсталляция
проще некуда:

=begin screen

    $ B<curl -LO http://xrl.us/perlbrew>
    $ B<chmod +x perlbrew>
    $ B<./perlbrew install>
    $ B<perldoc App::perlbrew>

=end screen

X<CPAN; local::lib>
X<CPAN; App::local::lib::helper>

Дистрибутив CPAN C<local::lib> позволяет вам устанавливать и управлять
дистрибутивами в вашем домашнем каталоге, чем системными в целом. Это
эффективный путь для сопровождения CPAN дистрибутивов без затрагивания прочих
пользователей в системе. Установка несколько сложнее, чем предыдущие два
дистрибутива, хотя C<App::local::lib::helper> может упростить этот процесс.
Смотрите U<http://search.cpan.org/perldoc?local::lib> и
U<http://search.cpan.org/perldoc?App::local::lib::helper> для деталей.

Все три проекта подразумевают Unix-подобное окружение (такие как дистрибутивы
GNU/Linux или даже Max OS X). Пользователи Windows могут посмотреть страницу
загрузки Padre всё-в-одном (U<http://padre.perlide.org/download.html>).
